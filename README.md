# Coolapk UWP
一个基于UWP的酷安客户端

基于[@一块小板子](http://www.coolapk.com/u/695942)的源码([Github链接](https://github.com/oboard/CoolApk-UWP))
## 现有功能
1. 浏览和下载应用
2. 访问开发人员中心
3. 浏览头条贴
4. 登录帐号
5. 夜间模式

## 如何安装
### 最低需求
- Windows 10 Build 15063及以上
- 设备需支持ARM/ARM64/x86/x64
- 至少50MB的空余储存空间(用于储存安装包与安装应用)
